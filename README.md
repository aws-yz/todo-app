# 🍎 提醒事项 - Apple-style Todo App

简洁优雅的待办事项管理应用，支持 PWA 离线使用。

## ✨ 特性

- 🍎 Apple 风格 UI 设计
- 🌙 自动深色模式
- 💾 LocalStorage 本地存储
- 📱 PWA 支持（可添加到桌面）
- 🔌 离线可用
- ⚡ Vite + React 构建

## 🚀 在线体验

访问: https://aws-yz.github.io/todo-app

## 📱 添加到桌面

**iOS Safari:**
1. 打开网址
2. 点击分享按钮
3. 选择 "添加到主屏幕"

**Android Chrome:**
1. 打开网址
2. 点击菜单 → "添加到主屏幕"

## 🛠️ 本地开发

```bash
npm install
npm run dev
```

## 📦 构建

```bash
npm run build
```
